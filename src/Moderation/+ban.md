# +ban
Bans a user with optional reason

## Usage
```
+ban <user> {reason}
```

## Arguments
Name | Description | Type | Required
:-- | :-- | :-- | :--
user | The user to ban. | User | Yes
reason | The reason for the ban if any. | String | No
