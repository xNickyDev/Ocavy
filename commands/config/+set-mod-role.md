# +set-mod-role
Set up a moderator role

> **Note** : With the moderator role you gain members access to moderation related commands and features.

## Usage
```
+set-mod-role <role>
```

## Arguments
Name | Description | Type | Required
:-- | :-- | :-- | :--
role | The moderator role to set. | Role | Yes
