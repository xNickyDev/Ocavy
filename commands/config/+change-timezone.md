# +change-timezone
Changes the timezone of the bot

## Usage
```
+change-timezone <timezone>
```

## Arguments
Name | Description | Type | Required
:-- | :-- | :-- | :--
timezone | The new timezone. | Timezone | Yes

> **Note** : Use `+timezones` to get a list of all available timezones.
