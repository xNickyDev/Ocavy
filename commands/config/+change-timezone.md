# +change-timezone

## Usage

```
+change-timezone <timezone>
```

## Arguments

| Name     | Description       | Type     | Required |
| -------- | ----------------- | -------- | -------- |
| timezone | The new timezone. | Timezone | Yes      |

{% hint style="info" %}
Use `+timezones` to get a list of all available timezones.
{% endhint %}
