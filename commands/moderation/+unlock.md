# +unlock

## Usage

```
+unlock {channel}
```

## Arguments

| Name    | Description            | Type    | Required |
| ------- | ---------------------- | ------- | -------- |
| channel | The channel to unlock. | Channel | No       |

> **Note** : When no channel is specified the current channel will be unlocked.
