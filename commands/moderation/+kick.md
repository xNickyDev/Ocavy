# +kick

## Usage

```
+kick <user> {reason}
```

## Arguments

| Name   | Description                     | Type   | Required |
| ------ | ------------------------------- | ------ | -------- |
| user   | The user to kick.               | User   | Yes      |
| reason | The reason for the kick if any. | String | No       |
