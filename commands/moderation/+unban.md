# +unban
Unbans a user with optional reason

## Usage
```
+unban <userID> {reason}
```

## Arguments
Name | Description | Type | Required
:-- | :-- | :-- | :--
userID | The user to unban. | User | Yes
reason | The reason for the unban if any. | String | No
