# +pin
Pins a message

## Usage
```
+pin <messageID> <channel>
```

## Arguments
Name | Description | Type | Required
:-- | :-- | :-- | :--
messageID | The message to pin. | Message | Yes
channel | The channel of the message to pin. | Channel | Yes
