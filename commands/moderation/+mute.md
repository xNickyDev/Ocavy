# +mute

## Usage

```
+mute <user> <duration> {reason}
```

### Aliases

```
+timeout
```

## Arguments

| Name     | Description                     | Type         | Required |
| -------- | ------------------------------- | ------------ | -------- |
| user     | The user to mute.               | User         | Yes      |
| duration | The duration of the mute.       | Alphanumeric | Yes      |
| reason   | The reason for the mute if any. | String       | No       |
