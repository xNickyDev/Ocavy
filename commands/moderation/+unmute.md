# +unmute
Unmutes a user with optional reason

## Usage
```
+unmute <user> {reason}
```

## Arguments
Name | Description | Type | Required
:-- | :-- | :-- | :--
user | The user to unmute. | User | Yes
reason | The reason for the unmute if any. | String | No
