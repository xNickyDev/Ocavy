# +clear
Clears messages

## Usage
```
+clear <amount> {user}
```

## Arguments
Name | Description | Type | Required
:-- | :-- | :-- | :--
amount | The amount of messages to clear. | Number | Yes
user | The user to clear messages from. | User | No
