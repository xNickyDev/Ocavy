# +unpin

## Usage

```
+unpin <messageID> <channel>
```

## Arguments

| Name      | Description                          | Type    | Required |
| --------- | ------------------------------------ | ------- | -------- |
| messageID | The message to unpin.                | Message | Yes      |
| channel   | The channel of the message to unpin. | Channel | Yes      |
