# +warn
Warns a user with optional reason

## Usage
```
+warn <user> {reason}
```

## Arguments
Name | Description | Type | Required
:-- | :-- | :-- | :--
user | The user to warn. | User | Yes
reason | The reason for the warn if any. | String | No
