# +steal-emoji

## Usage

```
+steal-emoji <emoji> {name}
```

### Aliases

```
+steal
```

## Arguments

| Name  | Description                           | Type   | Required |
| ----- | ------------------------------------- | ------ | -------- |
| emoji | The emoji to steal.                   | Emoji  | Yes      |
| name  | The name for the newly created emoji. | String | No       |
