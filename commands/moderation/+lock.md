# +lock

## Usage

```
+lock {channel}
```

## Arguments

| Name    | Description          | Type    | Required |
| ------- | -------------------- | ------- | -------- |
| channel | The channel to lock. | Channel | No       |

{% hint style="info" %}
When no channel is specified, the current channel will be locked.
{% endhint %}
