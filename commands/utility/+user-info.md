# +user-info

## Usage

```
+user-info {user}
```

### Aliases
```
ui
user
userinfo
```

## Arguments

| Name | Description                           | Type | Required |
| ---- | ------------------------------------- | ---- | -------- |
| user | The user to receive information from. | User | No       |
