# +color-info

## Usage

```
+color-info <hex/role>
```

### Aliases
```
+color
+colorinfo
```

## Arguments

| Name               | Description                             | Type               | Required |
| ------------------ | --------------------------------------- | ------------------ | -------- |
| <p>hex<br>role</p> | The color to receive information about. | <p>Hex<br>Role</p> | Yes      |

{% hint style="info" %}
If you specify a role, the color information about the given role's color is returned.
{% endhint %}
