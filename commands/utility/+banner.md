# +banner

## Usage

```
+banner {user}
```

### Aliases

```
+user-banner
```

## Arguments

| Name | Description                      | Type | Required |
| ---- | -------------------------------- | ---- | -------- |
| user | The user to get the banner from. | User | No       |
