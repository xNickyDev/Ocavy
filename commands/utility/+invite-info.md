# +invite-info
Returns information about the specified invite

## Usage
```
+invite-info <invite link/code>
```

## Arguments
Name | Description | Type | Required
:-- | :-- | :-- | :--
invite link <br>code | The invite from which to receive information. | Invite | Yes
