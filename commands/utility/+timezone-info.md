# +timezone-info

## Usage

```
+timezone-info <timezone>
```

## Arguments

| Name     | Description                               | Type     | Required |
| -------- | ----------------------------------------- | -------- | -------- |
| timezone | The timezone to receive information from. | Timezone | Yes      |

{% hint style="info" %}
Use `+timezones` to get a list of all available timezones.
{% endhint %}
