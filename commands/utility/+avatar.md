description: Returns the avatar of a user

# +avatar

## Usage

```
+avatar {user}
```

### Aliases

```
+av
+user-avatar
```

## Arguments

| Name | Description                      | Type | Required |
| ---- | -------------------------------- | ---- | -------- |
| user | The user to get the avatar from. | User | No       |
