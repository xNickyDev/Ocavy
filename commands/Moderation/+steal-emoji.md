# +steal-emoji
Creates the specified emoji on the current server

## Usage
```
+steal-emoji <emoji>
```

### Aliases
```
+steal
```

## Arguments
Name | Description | Type | Required
:-- | :-- | :-- | :--
emoji | The emoji to steal. | Emoji | Yes
