# +lock
Locks a channel

## Usage
```
+lock {channel}
```

## Arguments
Name | Description | Type | Required
:-- | :-- | :-- | :--
channel | The channel to lock. | Channel | No

> ℹ️ When no channel is specified the current channel will be locked.
