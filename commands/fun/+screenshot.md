# +screenshot

## Usage

```
+screenshot <link>
```

### Aliases

```
+ss
```

## Arguments

| Name | Description                            | Type | Required |
| ---- | -------------------------------------- | ---- | -------- |
| link | The link of the website to screenshot. | URL  | Yes      |
