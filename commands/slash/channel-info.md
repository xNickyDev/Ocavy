# /channel-info

## Usage

```
/channel-info (channel)
```

## Arguments

| Name    | Description                              | Type    | Required |
| ------- | ---------------------------------------- | ------- | -------- |
| channel | The channel to receive information from. | Channel | No       |

{% hint style="info" %}
If no channel is specified, the current channel's information will be returned.
{% endhint %}
