# +untimeout
Untimeouts a user with optional reason

## Usage
```
+untimeout <user> {reason}
```

## Arguments
Name | Description | Type | Required
:-- | :-- | :-- | :--
user | The user to untimeout. | User | Yes
reason | The reason for the untimeout if any. | String | No
