# +check-warns
Checks the warnings of a user

## Usage
```
+check-warns {user}
```

### Aliases 
```
+warns
+warnings
````

## Arguments
Name | Description | Type | Required
:-- | :-- | :-- | :--
user | The user to check the warnings from. | User | No
